# Smart Bus Tracking System

## Overview
A comprehensive web-based solution for real-time bus tracking and management, enabling users to monitor bus locations, check schedules, and book tickets efficiently.

## Features
- **Real-time Tracking**: Monitor bus locations in real-time on an interactive map
- **Schedule Management**: View and manage bus schedules
- **Booking System**: Easy ticket booking and reservation
- **Driver Dashboard**: Dedicated interface for bus drivers
- **User Authentication**: Secure login system for passengers and drivers
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Installation
1. Clone this repository
2. Open `Main.html` in your web browser
3. Navigate through the application using the menu

## Usage
### For Passengers:
1. Login with your credentials
2. Search for available buses
3. Book tickets through the card interface
4. Track your bus in real-time

### For Drivers:
1. Login to driver dashboard
2. View assigned routes and schedules
3. Update bus status and location
4. Manage passenger information

## File Descriptions

- **login.html.html**: User authentication interface
